[TO RUN THE DEV SERVERS]
SNIPPETS:
run backend:

cd backend
.\venv\Scripts\activate
python main.py 

run frontend:
by building (production build)

Build:
cd frontend
flutter build web --release

Run:
cd frontend/build/web
python -m http.server 8888

Direct run: (debug)
flutter run -d edge

[Hosting]

[RENDER]
Simply push changes and then deploy manually from that commit 

[FIREBASE]
cd frontend
flutter build web --dart-define=SMARTMESS_BACKEND_URL=https://smartmess-lx1a.onrender.com

firebase deploy --only hosting
